Информация

Шаблон для сайта изначально был подобран для клиента, который вскоре передумал делать сайт.

Поскольку мне интересно было сделать такой лендинг, заодно освоив на нем некоторые новые для меня инструменты, я взялся за реализацию - просто для себя.

Некоторые особенности:

  • Решено было сделать сайт, максимально используя возможности фреймворка Bootstrap v3.3.5
  • Иконочный шрифт: fontawesome
  • Для анимаций была подключена библиотека animate.css, для анимации были прописаны свои тайминги
  • Для задания высоты границы срабатывания анимации, была подключена библиотека wow.js
  • Плавный скролл с ease-in-out эффектом был написан самостоятельно, на чистом JS (интересно было написать без использования jQuery и подключения готовых либ)
  • Сайт посажен на CMS Joomla! последней версии, дополнительные плагины не подключались
